# Dev-s-ConsoleRPG

A little text-based RPG that I made for a little project! I hope you enjoy it!!

## Features

- A cool class system
- Interesting Scenarios
- Skill choices based on class
- Save/Load for later

## Updates

### 0.1

- Created Class structures and displays for starting classes
- Greet the player along with giving the player choice on classes

## How To Build

- cd Dev-s-ConsoleRPG/src
- g++ Player.cpp main.cpp -o Devs_Console_RPG

## Usage

- ./Dev-s-ConsoleRPG

## Initialize Git Repository

- cd Dev-s-ConsoleRPG
- git init
- git add .
- git commit -m "Initial project structure"
