# Dev-s-ConsoleRPG

A little text-based RPG that I made for a little project! I hope you enjoy it!!

## Features

- A cool class system
- Interesting Scenarios
- Skill choices based on class
- Save/Load for later

## Updates

### 0.2

## How To Build

- cd Dev-s-ConsoleRPG/src
- g++ -std=c++20 main.cpp Player.cpp Enemy.cpp Battle.cpp -o consoleRPG

## Usage

- ./consoleRPG

- cd Dev-s-ConsoleRPG
